📦 Fs-Stash - Geavanceerd Kluis Systeem voor FiveM
Een lichtgewicht, prop-vrij en onzichtbaar kluissysteem voor ESX, volledig gebouwd rondom de moderne OX-stack. Laat spelers hun eigen geheime kluizen plaatsen, verkopen aan andere spelers via een tablet, en veilig beheren via ox_target.

✨ Functionaliteiten
Prop-Vrij & Lag-Vrij: Er worden geen 3D-objecten gespawnd. De kluis is een onzichtbare zone die alleen interactief is via ox_target (het "oogje").

Tablet Verkoopsysteem: Spelers kunnen het item stash_tablet gebruiken om een kluis op hun locatie aan te maken en deze direct te verkopen aan een andere speler in de buurt via een interactief menu.

Database Opgeslagen: Alle kluizen worden direct opgeslagen in de database en feilloos ingeladen na een server herstart. Beveiligd tegen corrupte data.

Eigendomsrechten: Alleen de speler die de kluis heeft gekocht (de eigenaar) krijgt de optie om de kluis te verwijderen.

Admin Bypass: Staff-leden (admin/superadmin) hebben de mogelijkheid om elke kluis te verwijderen, mocht deze in de weg staan of misbruikt worden.

Real-time Synchronisatie: Als een kluis wordt verwijderd, verdwijnt de interactie direct voor alle spelers op de server zonder dat een herstart nodig is.

🛠️ Vereisten (Dependencies)
Zorg dat de volgende resources geïnstalleerd en up-to-date zijn op je server:

es_extended (ESX Legacy)

ox_lib

ox_target

ox_inventory

oxmysql
